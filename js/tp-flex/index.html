Background:

El Módulo de FlexBox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los elementos de un contenedor y mejorar las capacidades de alineación. Cuando describimos a flexbox como unidimensional destacamos el hecho que flexbox maneja el layout en una sola dimensión a la vez (ya sea como fila o como columna). Esto contrasta con el modelo bidimensional del "Grid Layout de CSS", el cual controla columnas y filas a la vez.

Contenedores e Items:

El uso de flex conlleva a tener un Contenedor "flexible", al cual se le debe asignar "display: flex", los hijos directos de este contenedorseran los Items "flexibles", con solo hacer esto ya podemos observar el comoprtamiento por defecto, pero nosotros podemos sobreescribir dichos comportamientos para ajustarlos a nuestras necesidades, todo esto a travez de las propiedades que tendra el Contenedor Flexible y los Items Flexible.

Ejes:

Flex define un eje principal que es llamado "flex-direction" y otro que sera ortogonal a la dirección de este que es conocido como el eje cruzado. En otras palabras, flex-direction mostrará la dirección en la cual el vector irá incorporando cada nuevo item en el contenedor. Caracteristicas:

  1. Del Contenedor de Items

    • display flex: Convierte a un elemento en un Contenedor Flexible.
    • flex-direction: Indica la direccion en la que se acomodaran los Items, por defecto estaran en una sola fila. 
          .container {
        flex-direction: row | row-reverse | 
                        column | column-reverse;
    }
    • flex-wrap: Por defecto los Items se ajustan a una sola linea, con esta propiedad le podemos decir que de no entrar se utilice otra. 
          .container {
        flex-wrap: nowrap | wrap | wrap-reverse;
    }
    • flex-flow: Es la abreviación de las propiedades Flex-Direction y Flex-Wrap que en conjunto definen la flexibilidad del contenedor y sus ejes. El primer valor del atributo indica la dirección y el segundo el wrap. 
          .container {
        flex-flow: column wrap;
    }
    • justify-content: Define como se distribuira el espacio libre del contenedor en el eje principal, por defecto este espacio va al final 
          .container {
        justify-content: flex-start | flex-end | 
                        center | space-between | 
                        space-around | 
                        space-evenly | 
                        start | end | 
                        left | right;
    }
    • align-items: Define para todos los items como se alineara con respecto al eje secundario, por defecto toma la misma medida que tiene el contenedor 
          .container {
        align-items: stretch | flex-start | 
                    flex-end | center | 
                    baseline | 
                    start | end;
    }
    • align-content: Define como distribuira el espacio libre del contenedor en el eje secundario, esto solo tiene efecto cuando se tiene multilineas 
          .container {
        align-content: flex-start | flex-end | 
                       center | space-between | 
                       space-around | 
                       space-evenly | 
                       stretch | start | 
                       end | baseline | 
    }

  2. Del Item

    • order: Define en que posicion del contenedor se pondra el elemento, por defecto se respeta el orden de entrada 
          .item {
        order: 5; /* default 0 */
    }
    • flex-grow y flex-shrink: Flex Grow define la capacidad de crecer(en el eje principal) a la hora de distribuir el espacio libre del contenedor, mientras mas alto mas espacio restante tomara. Flex Shrink define la capacidad de achicarse(en el eje principal) 
          .item {
        flex-grow: 4; /* default 0 */
    }
                        
          .item {
        flex-shrink: 3; /* default 1 */
    }
    • flex-basis: Define el tamaño por defecto del elemento(en el eje principal) antes de que el espacio libre sea distribuido 
          .item {
        flex-basis:  | auto; /* default auto */
    }
    • flex: Es la abreviacion de las propiedades Flex-Grow, Flex-Shrink y Flex-Basis
    • align-self: Define como se alineara el Item con respecto al eje secundario 
          .item {
        align-self: auto | lex-start | 
                    flex-end | center |
                    baseline | stretch;
    }
